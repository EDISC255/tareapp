# TAREA

## ARQUITECTURA M-V-C O B-C-E
ESTE PROYECTO UTILIZARA LA ARQUITECTURA MODEL-VIEW-CONTROL O BORDE-CONTROL-ENTIDAD, ES UNA ARQUITECTURA DE PROGRAMACION QUE SE CARACTERIZA EN LA SEPARACIONN DEL CODIGO EN 3 PARTES O DIRECTORIOS (*BORDE O VISTA, CONTROL Y ENTIDAD O MODELO*) PARA UNA MEJOR GESTION DEL CODIGO.

* *ENTIDAD | MODELO*: LAS CLASES ENTIDAD SALEN DE LA BASE DE DATOS EN LA SE ESTA TRABAJANDO.

* *CONTROL*: EN LAS CLASES CONTROL SE IMPLEMENTA LA LOGICA DE PROGRAMACION PARA CADA ENTIDAD.

* *BORDE O VISTA*: EN LA CLASES BORDE SE IMPLEMENTA EL DISEÑO DE LAS INTERFACES GRAFICAS DE USUARIO. 

ESTA ARQUITECTURA ES USADA PARA LA PROGRAMACION PARA ESCRITORIO PRINSIPALMENTE
